# TitanicPrediction
Проект по анализу данных и машинному обучении модели на основе датасета Титаника.

Данный проект базируется на [датасете выживших "Титаника"](https://www.kaggle.com/datasets/yasserh/titanic-dataset).

Документация к коду доступна по [этой ссылке](https://alexeev-prog.github.io/TitanicPrediction/).

Онлайн-сервис доступен по [этой ссылке](https://titanic156.streamlit.app/).

## О проекте

Полнофункциональное веб-приложение для прогнозирования выживаемости пассажиров Титаника, реализующее полный цикл Data Science проекта: от анализа данных и предобработки до обучения модели и развертывания в production.

**Ключевые особенности:**
-  **Кастомная реализация** логистической регрессии с градиентным спуском
-  **Интерактивный EDA** с визуализацией закономерностей выживаемости
-  **Объяснимый ИИ** - анализ факторов, влияющих на предсказания
-  **Production-ready** веб-интерфейс с Streamlit
-  **Чистая архитектура** с соблюдением SOLID принципов

## Набор данных

Проект использует классический [Titanic Dataset с Kaggle](https://www.kaggle.com/datasets/yasserh/titanic-dataset), содержащий информацию о 891 пассажире:

**Признаки:**
- `Pclass` - класс билета (1, 2, 3)
- `Sex` - пол пассажира
- `Age` - возраст
- `SibSp` - количество братьев/сестер/супругов на борту
- `Parch` - количество родителей/детей на борту
- `Fare` - стоимость билета
- `Embarked` - порт посадки
- `Cabin` - номер каюты
- `Ticket` - номер билета

**Целевая переменная:**
- `Survived` - выжил (1) или нет (0)

## Быстрый старт

### 1. Клонирование репозитория
```bash
git clone https://github.com/alexeev-prog/TitanicPrediction.git
cd TitanicPrediction
```

### 2. Установка зависимостей
```bash
# Используя UV (рекомендуется)
uv sync

# Или используя pip
pip install -r requirements.txt
```

### 3. Запуск приложения
```bash
# Основной скрипт
python main.py

# Или через launcher
python run_app.py --port 8501 --host localhost
```

### 4. Открытие в браузере
```
http://localhost:8501
```
